# ADR-005: BallTree 공간 인덱스로 거리 계산

- **상태**: Accepted
- **날짜**: 2026-03-20

## 맥락

16,755개 아파트와 182,902개 시설 간 5km 이내 거리를 계산해야 한다. 브루트 포스로 모든 쌍을 계산하면 30억 쌍이 되어 비현실적이다.

## 결정

scikit-learn의 `BallTree` (haversine metric)를 사용하여 아파트별 5km 반경 내 시설을 효율적으로 검색하고 거리를 계산한다.

## 근거

- **효율성**: BallTree는 O(n log n) 시간 복잡도로 반경 검색을 수행한다. 브루트 포스 O(n*m) 대비 획기적으로 빠르다.
- **정확도**: haversine 거리 메트릭으로 지구 곡면을 고려한 실제 거리(m)를 계산한다.
- **검증된 구현**: scikit-learn의 안정적인 구현을 활용한다.

## 결과

- 4,900만 행의 `apt_facility_mapping` 테이블 생성 (아파트-시설 쌍 + 거리).
- 데이터 파이프라인 `step4_calc_distance.py`에서 빌드 타임에 한 번 실행.
- 이후 서비스에서는 사전 계산된 거리 테이블을 쿼리한다.
